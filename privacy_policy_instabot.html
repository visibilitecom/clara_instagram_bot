Politique de Confidentialité

Bienvenue sur le bot Instagram de Clara. Cette politique explique comment nous utilisons les informations échangées via Instagram.

1. Collecte de données

Nous collectons uniquement les messages que vous nous envoyez via Instagram pour pouvoir vous répondre de manière personnalisée.

2. Utilisation des données

Les données sont utilisées uniquement pour générer des réponses automatiques grâce à une intelligence artificielle (OpenAI). Aucune donnée n’est utilisée à des fins commerciales ou revendues.

3. Stockage

Vos messages peuvent être temporairement stockés pour améliorer l’expérience conversationnelle. Ils sont automatiquement supprimés après un certain temps.

4. Suppression des données

Si vous souhaitez que vos données soient supprimées immédiatement, envoyez un message à : claradimigli@gmail.com

5. Consentement

En envoyant un message au bot Instagram, vous acceptez cette politique de confidentialité.
