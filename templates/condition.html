Conditions de service

Bienvenue sur le service Clara Instagram Bot.

1. Description du service

Ce service permet d’interagir avec un agent conversationnel automatisé via Instagram. Les réponses sont générées par intelligence artificielle à partir des messages reçus.

2. Données collectées

Nous collectons uniquement les données nécessaires à l’interaction avec le bot : identifiant Instagram, contenu des messages, et date d’envoi. Ces données sont stockées temporairement pour améliorer la conversation.

3. Utilisation des données

Les données sont utilisées exclusivement pour fournir une réponse personnalisée via le bot. Aucune donnée n’est vendue ou partagée à des tiers.
